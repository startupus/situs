## 🎯 Технический стек

- **База данных**: PostgreSQL + Prisma ORM
- **API**: Express.js + TypeScript
- **Валидация**: Joi
- **Аутентификация**: JWT
- **Тестирование**: Vitest + Supertest
- **Документация**: Swagger/OpenAPI
- **Контейнеризация**: Docker
